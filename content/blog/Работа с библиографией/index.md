# Работа с библиографией

Работа с библиографией — важная часть любой научной или учебной деятельности. Грамотно оформленные источники показывают уровень подготовки автора, подтверждают достоверность информации и помогают избежать плагиата.

## Что такое библиография

Библиография — это список использованных источников: книг, статей, сайтов и других материалов, на которые опирается автор при написании работы.

## Зачем она нужна

- подтверждает аргументы и факты  
- показывает глубину исследования  
- помогает читателю найти первоисточники  
- защищает от обвинений в плагиате  

## Основные виды источников

- Книги  
- Научные статьи  
- Электронные ресурсы  
- Диссертации и авторефераты  
- Нормативные документы  

## Популярные стандарты оформления

Существует несколько стандартов оформления библиографии:

- ГОСТ (часто используется в странах СНГ)  
- APA (распространён в международных работах)  
- MLA  
- Chicago  

Важно использовать **один стиль оформления** во всей работе.

## Пример оформления

### Книга
Иванов И.И. *Основы научных исследований*. — Москва: Наука, 2020.

### Статья
Петров П.П. Современные методы анализа данных // Научный журнал. — 2021. — №3. — С. 45–50.

### Интернет-ресурс
Название сайта. URL: https://example.com (дата обращения: 01.01.2026).

## Полезные инструменты

Для упрощения работы с источниками можно использовать специальные сервисы:

- Google Scholar — для поиска научных статей  
- eLIBRARY — для российских научных публикаций  
- Zotero / Mendeley — для хранения и оформления ссылок  

## Советы

- Записывайте источники сразу, не откладывайте  
- Проверяйте корректность оформления  
- Используйте только надёжные источники  
- Следите за актуальностью информации  

## Заключение

Грамотная работа с библиографией — это не формальность, а важный навык, который влияет на качество и доверие к вашей работе. Освоив основные правила и инструменты, можно значительно упростить процесс подготовки научных текстов.
