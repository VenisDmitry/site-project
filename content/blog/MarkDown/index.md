# Markdown

Markdown — это легковесный язык разметки, который используется для оформления текста с помощью простого синтаксиса.

## Где используется
- GitHub
- Документация
- Блоги
- README файлы

## Основные возможности

### Заголовки
# Заголовок 1
## Заголовок 2
### Заголовок 3

### Форматирование текста
**жирный текст**  
*курсив*  
~~зачёркнутый текст~~

### Списки

Маркированный:
- пункт 1
- пункт 2
- пункт 3

Нумерованный:
1. первый
2. второй
3. третий
